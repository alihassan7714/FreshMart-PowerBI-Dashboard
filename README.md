# FreshMart Power BI Dashboard

## Project Overview

Built an interactive Power BI dashboard for a retail supermarket chain. Cleaned and transformed sales data using Power Query, created DAX KPIs (Revenue, Profit, Orders, AOV, Profit Margin), and designed interactive visualizations for sales trends, product performance, customer segments, and regional analysis.
